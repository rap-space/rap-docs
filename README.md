### 环境准备



1. 安装 Node

Node.js 是一个开放源代码、跨平台的、可用于服务端和网络应用的运行环境。rap-cli 使用了 Node.js 开发，因此需要需要开发者安装 [Node.js](https://nodejs.org/en/) 。



2. 设置淘宝镜像[可选]  为你的 npm 安装加速

```bash
$ npm config set registry https://registry.npm.taobao.org
```


3. 安装rap-cli
